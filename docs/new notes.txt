regs:

	12 for extended sprites (120 bytes)

	21 for bounce sprites (84 bytes)

	7 for shooters (56 bytes)

	8 for coin sprites (32 bytes)

	7 for score sprites (42 bytes)

	10 for minor extended sprites (120 bytes)

	4 for smoke sprites (16 bytes)

	(470 bytes in total)


	Each ExSprite gets 13 regs:

	- xpos lo
	- xpos hi
	- xspeed
	- xfraction
	- ypos lo
	- ypos hi
	- yspeed
	- yfraction

	- GFX data
	- go behind scenery

	- misc 1
	- misc 2
	- misc timer

	Bounce sprites will need some rewriting but that's fine.

$1699-$17BA:	290 (0x122) bytes

$17C0-$185B:	156 (0x09C) bytes


	- Physics -
	$1699: X position lo byte
	$16B9: X position hi byte
	$16D9: X speed
	$16F9: X fraction bits
	$1719: Y position lo byte
	$1739: Y position hi byte
	$1759: Y speed
	$1779: Y fraction bits

	- Misc -
	$1799: Misc. timer
	$17C0: Misc. 1
	$17E0: Misc. 2
	$1800: GFX data
	$1820: Go behind scenery flag (7 bits free...)
